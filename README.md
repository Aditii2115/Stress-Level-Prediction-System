- Data visualization using bar charts

- Exception handling for file operations

Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Object-Oriented Programming (OOP)

Machine Learning Models

The following models were implemented and compared:

- Logistic Regression
- Decision Tree
- Random Forest 

The Random Forest model achieved the highest accuracy (~0.70) and was selected as the best model for prediction.

- Train multiple machine learning models
- Compare model accuracy
- Visualize results
- Predict stress level using user input
